Help

Navigation:

Navigation is simple. There are five icons located on the main portal pages.

Home
Apps
Games
Net Apps
System Info

Click on an icon to be taken to the appropriate page. On each page are listed text links to the respective content.

Note that Net Apps are all internet locations and require a Wi-Fi connection to access.

Running Other Portals:

You can easily run any other portal that's located in the COMMON folder. Just go to the home page of PSP Optimize, then use the Run File feature to easily navigate to the other portal's home page. This is usually directly in the portal's folder and called "index.html" or "home.html." Run the file, and the other portal will launch.

Using the PDF/Word Viewer:

To use the PDF/Word Viewer, you must click on the "View document here" link that shows up after submitting a file. If you miss it, the page will redirect to a error. You can go back by using the L-trigger and try again.

Text Messaging:

Text messaging service is provided by txt2day.com. This service provides the ability to receive text messages by entering your E-mail address at the time you send a message. Any replies to your message will be sent to your E-mail address. To quickly access a Gmail, Hotmail, or Yahoo account, use the E-mail links provided in Net Apps.

Using SkreemR:

The PSP does not have enough memory to use SkreemR's built-in flash player to play mp3 files. You need to save the file to your Memory Stick by clicking on the name of the file. Generally a search term consisting of a song name and the artist name will yield a result.

Customizing the background:

The background of PSP Optimize can be easily customized to any image.

1) Ensure that your chosen image has the dimensions 480x272.

2) Place the image into the images/background folder inside the main pspoptimize folder.

3) Find "background.png" inside the pspoptimize.css file and replace it with the name of your image.

Text colors can also be changed. To change text color, edit the .text color. For headings, edit .heading. For links, edit a:link and a:visited.

Updating:

You can find the latest edition of PSP Optimize at http://pspoptimize.summacentral.com. For convenience, you can download the update .zip file directly to your PSP. An update link is included in the Info page to do this. Once you can access a computer, connect your PSP and unzip the file, then follow typical installation instructions, located in the readme file.

Note that you will lose your customized background and text colors after an update. Changes will need to be reapplied.
